## 三、功能实现

### 3.1 已实现功能

| 功能 | 状态 |
|------|------|
| 普通订单创建 | ✅ |
| VIP 订单创建 | ✅ |
| VIP 优先级（排在普通订单前） | ✅ |
| 订单编号唯一递增 | ✅ |
| 添加机器人 | ✅ |
| 机器人自动处理订单 | ✅ |
| 10 秒处理时间 | ✅ |
| 机器人 IDLE 状态 | ✅ |
| 移除机器人 | ✅ |
| 订单返回队列（保持优先级） | ✅ |
| 无数据持久化 | ✅ |
| HH:MM:SS 时间戳输出 | ✅ |
| 交互式 CLI | ✅ |

### 3.2 命令支持

| 命令 | 功能 |
|------|------|
| `normal` | 创建普通订单 |
| `vip` | 创建 VIP 订单 |
| `add-bot` | 添加机器人 |
| `remove-bot` | 移除机器人 |
| `status` | 显示系统状态 |
| `exit` | 退出 |

## 五、部署方式

### 5.1 GitHub Actions
- `script/test.sh` - 运行单元测试
- `script/build.sh` - 编译 Go 程序
- `script/run.sh` - 运行程序
- `result.txt` - 包含时间戳输出

### 5.2 本地运行
```bash
./script/build.sh    # 编译
./bin/mcdonald       # 运行
```
